# 最初の一歩 Director

## チャンネル概要
- **名前**: 最初の一歩 — 動けなかったあなたへ
- **コンセプト**: 金融×モチベーション。知識はあるけど動けない人の背中を押す
- **ターゲット**: 20-40代、投資未経験〜初心者
- **形式**: faceless長尺（10-15分目標）、毎日投稿
- **トーン**: 説教ではなく寄り添い。「大丈夫、これだけやればいい」

## 制作フロー

### 1. 台本作成
`episodes/{episode_id}/script.txt` を作成する。

**フォーマット**:
```
ナレーション文 | テーマラベル
```

**ルール**:
- 1行 = 1シーン（1カット）
- `|` の左: ナレーションの全文（音声として読まれる）
- `|` の右: 画面に表示されるテーマラベル（字幕ではない。その場面のテーマを短く）
- 最初の行: 導入（視聴者の共感を得るフック）
- 最後の行: クロージング（行動を促すメッセージ）
- 15-20行が目安（10-15分の動画）

**台本の構成パターン**:
1. **共感フック**: 「〜で悩んでいる。そんなあなたへ」
2. **問題提起**: なぜ動けないのか、なぜわからないのか
3. **核心の知識**: 1つだけ覚えてほしい概念
4. **具体例+数字**: データで説得
5. **反論処理**: 「でも〜だったら？」→ 安心材料
6. **行動の提示**: やることは1つだけ
7. **背中を押す**: 完璧じゃなくていい
